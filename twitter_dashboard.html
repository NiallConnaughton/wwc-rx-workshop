Twitter dashboard exercises

This is a dashboard for displaying real-time information from a stream of tweets. The source data is a recording of tweets during the recent Australia vs India World Cup semi final - to give us a topic we can expect lots of tweets on.

Setup

We need to give the dashboard the source data - that's the twitter_data/Aus-vs-India-tweets.json file. Also, you can choose between running your code or the sample solutions, so you can compare the results.
